Programming Skill

下請けが多いSIer企業だったので、プログラミング言語は数か月単位で変わってました. 広く浅く.
フロント/バックエンドの境なく開発. モダンな言語は業務では携われる機会がなく残念. (´・ω・｀)
社内SEになってからはクラウド化すべく、Google WorkspaceやGCPを使い始める。
ノーコードやRPAにもチャレンジ中。
その他 : PowerShell / Classic ASP / VBS / sh,batch / 学生時代はC, C++, HDL, TeX など.
